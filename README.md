# Rubber Duck

Virtual rubber duck. Say goodbye to private room retreats pretending you are on a call when you really just need to rubberduck something.

- Only HTML, CSS and SVG
- No JavaScript
- Independent on local/remote files
- Small size
- Simple design

Installation and usage instructions in case you never used a rubber duck or you cannot imagine how to use a virtual one.

1. Download either `ducky.html` (good for editing or studying the immense beyond-complex source code) or `ducky.min.html` (for <q>production</q> use)
2. (Recommended) Verify SHA256 checksum of the downloaded file
3. Open the downloaded file in your favourite browser (e.g. [Firefox](https://www.firefox.com/))
4. <abbr title="Explain Like I'm 5">ELI5</abbr> your issue to the *virtual* rubber duck as if it was your *real* rubber duck

Note: if you click on Submit, **nothing** happens (on purpose), however, you can disable Javascript to make sure. The button is there purely for aesthetic purposes.

That's it, folks.
